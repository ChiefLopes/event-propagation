Event Propagation

What is Event Propagation?

This has to do with how events travel through the DOM until it gets to its desired node target.

It is like the way an electric curent travels through a wire until it gets to its destination.

Since the DOM is full of nodes, an evnt has to travel or pas through these nodes until it gets to the point of target or its intended node destination.

There are 3 types of Event Propagation

  • Event Capturing Phase
  • Target Phase
  • Event Bubling Phase

Explanations:

  • The Capturing Phase ⇒ This is the phase where the event pases from the parent nodem and travels down to the child element where it is targetted
